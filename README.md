# linux_notes_hanshunping
> 基于 B 站韩顺平 Linux 视频教程，逐章整理命令、配置、脚本。  
> **补充了原视频缺失的指令格式，修正了过时/错误的内容**，所有操作均在 CentOS 7/8 上实际验证。

## 📚 内容涵盖

- 文件目录管理、权限、链接  
- 用户/组管理、运行级别、服务管理（service / systemctl）  
- 磁盘分区、挂载、`fdisk` 实操  
- 进程、网络、防火墙、定时任务（cron / at）  
- 软件包管理（rpm / yum / apt）  
- Shell 编程（变量、流程控制、函数、备份脚本）  
- 日志管理、内核源码、备份恢复（dump/restore）  
- Webmin / 宝塔面板安装  
- 自定义 Linux 系统（grub 引导、最小系统制作）

## ✨ 特点

- ✅ **指令格式补全** —— 原视频中省略的参数、选项均按 `man` 手册补全  
- ✅ **勘误与适配** —— 针对 CentOS 7→8、过时命令（如 `service`→`systemctl`）给出正确做法  
- ✅ **实操验证** —— 每条命令、每个配置文件都经过真实环境测试  
- ✅ **笔记即手册** —— 可直接当作查命令的快速参考

## ⚠️ 使用前注意

仓库中原始文件（`linux指令.md`）**包含作者测试时的真实 IP 和密码**，已做脱敏示例（如 `your_password`、`your_server_ip`）。  
如果你直接克隆使用，请务必自行替换所有敏感信息。

## 📖 如何阅读

- 按章节顺序阅读，大部分命令带有示例输出  
- 可直接搜索关键字（如 `crontab`、`firewall-cmd`）快速定位  
- 建议配合 [Linux man pages](https://man7.org/linux/man-pages/) 对照理解
